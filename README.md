 ToDoApp (Java Swing)
A simple desktop To‑Do List application built using Java Swing. 
This app allows users to add tasks and delete tasks interactively with a clean GUI.
Features
Add new tasks using the text field and ADD button.
Select a task and remove it using the DELETE button.
Scrollable task list for managing multiple items.
Lightweight, beginner‑friendly Java Swing implementation.
Project Structure
JFrame Main application window.
JTextField Input field for entering tasks.
DefaultListModel Stores tasks dynamically.
JList Displays tasks in a scrollable list.
JButton Buttons for ADD and DELETE actions.
ActionListener Handles button clicks.
How It Works
Launch the app A window titled TO DO APP opens.
Enter a task in the text field at the top.
Click ADD The task is added to the list below.
Select a task from the list.
Click DELETE The selected task is removed.
Repeat to manage your to‑do list interactively.
